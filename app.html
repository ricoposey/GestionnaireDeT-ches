Voici un aperçu des fonctionnalités disponibles :

1. Gestionnaire de Tâches

Dans cette section, vous pouvez ajouter, modifier, et supprimer vos tâches. Utilisez le formulaire pour entrer les détails de chaque tâche :

  • Nouvelle Tâche : Entrez le nom de votre tâche dans le champ "Nouvelle tâche".
  • Date d'échéance : Sélectionnez la date à laquelle la tâche doit être complétée.
  • Rappel : Cochez cette case si vous souhaitez recevoir une notification pour cette tâche.
  • Répéter : Cochez cette case si vous souhaitez que la tâche se répète à intervalles réguliers.
  • Catégorie : Choisissez une catégorie pour organiser vos tâches (Travail, Personnel, Etudes, Ménage).
  • Priorité : Sélectionnez le niveau de priorité de la tâche (Faible, Importante, Urgente).
  • Ajouter : Cliquez sur "Ajouter" pour enregistrer la tâche.
  • Mettre à jour : Cliquez sur "Mettre à jour" pour modifier une tâche existante.

2. Liste des Tâches

Cette section affiche toutes vos tâches. Vous pouvez voir les détails, les modifier ou les supprimer en utilisant les boutons correspondants.

3. Tâches Prioritaires

Affiche les tâches marquées comme "Urgentes". Cela vous permet de voir rapidement vos tâches les plus importantes.

4. Calendrier

Le calendrier vous permet de visualiser vos tâches en fonction de leurs dates d'échéance. Vous pouvez voir toutes vos tâches planifiées dans une vue mensuelle.

Menu Hamburger

Sur les petits écrans, utilisez le menu hamburger pour naviguer entre les différentes sections de l'application.
